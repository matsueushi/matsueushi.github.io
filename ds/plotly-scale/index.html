Plots.jlでx軸、y軸のスケールを揃える

2019年03月24日 データサイエンス
  • Plots
  • Julia

 aspece_ratio=1 とすれば良いだけだった。簡単だった。

using Distributions
using Plots
x = y = 0:0.5:5
exp_x = Exponential(3)
exp_y = Exponential(10)
z = Surface((s, t) -> pdf(exp_x, s) * pdf(exp_y, t), x, y)
heatmap(x, y, z)


heatmap(x, y, z, aspect_ratio = 1)
