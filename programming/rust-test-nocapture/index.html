Rustでcargo test中にprintlnする

2022年08月28日 · プログラミング

cargo testでテストを実行しているとき、出力はキャプチャされてしまってテスト実行画面に出てこないが、

$ cargo test -- --nocapture

で出てくるようにできるようだ。
