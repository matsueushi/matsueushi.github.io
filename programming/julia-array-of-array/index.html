JuliaでArray of Arrayを1次元Vectorにする方法

2019年03月20日 · プログラミング

よく忘れるのでメモ。

結論から言うと Base.Iterators.flatten を適用して collect すれば良い。

julia> import Base.Iterators: flatten
julia> xs = [[1, 2, 3], [4, 5, 6]]
2-element Array{Array{Int64,1},1}:
 [1, 2, 3]
 [4, 5, 6]
julia> collect(flatten(xs))
6-element Array{Int64,1}:
 1
 2
 3
 4
 5
 6

追記(2019/4/7):

 vcat を使った方が簡潔だった。

julia> vcat(xs...)
6-element Array{Int64,1}:
 1
 2
 3
 4
 5
 6
