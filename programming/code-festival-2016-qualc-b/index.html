AtCoder Code Festival 2016 Qual C B - K個のケーキ

2022年05月28日 · プログラミング
  • AtCoder

考察中

ケーキの数が多い順にソートして、なるべく前日と同じケーキを食べないように1種類目、 2種類目のケーキを交互に食べていく。そのうち2種類目のケーキがなくなるので、 1種類目のケーキと3種類目のケーキを交互に食べていく。 そのうち1種類目のケーキか3種類目のケーキがなくなるから、次に4種類目のケーキを交互に食べる対象とする、 という戦略。
